When It's Write

When It’s Write, an Android application that allows songwriters to capture ideas for their songs wherever they. Songwriters don’t always have a notepad or a computer to capture lyric ideas but that doesn’t stop the songs from coming. When It’s Write will allow songwriters to capture and organize lyrics while they are on the go.

When It’s Write allows users to log in through their Facebook account and uses Firebase, an online database, to store the lyrics so a user can access their songs that they saved from any Android device that has the app.

When It’s Write was created because you never know when creativity will strike and you need to be read because, When It’s Write, it’s right.

It can be found on my GitHub page at: https://github.com/acjones212/whenitswrite and on Google Play at: https://play.google.com/store/apps/details?id=com.austinjones.whenitswrite.
